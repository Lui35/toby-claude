# Quick Start Guide

## Installation (2 minutes)

1. **Open Chrome Extensions**
   - Type `chrome://extensions/` in your address bar and press Enter

2. **Enable Developer Mode**
   - Look for the toggle in the top-right corner
   - Switch it ON

3. **Load Extension**
   - Click "Load unpacked"
   - Browse to: `C:\Users\Lui-PC\Desktop\Reboot\Toby-Claude\extension`
   - Click "Select Folder"

4. **Done!**
   - Extension icon appears in toolbar
   - Open a new tab to see the dashboard

## First Steps

### Create Your First Collection
1. Open a new tab
2. Click "New Collection"
3. Name it (e.g., "Work", "Research", "Shopping")
4. Click "Create"

### Add Tabs to Collection
1. Navigate to a website you want to save
2. Click the extension icon in toolbar
3. Click on the collection name
4. Tab is now saved!

### Import Your Toby Data
1. Open new tab (dashboard)
2. Click "Import" button
3. Select your Toby export file: `toby-collections-2025-11-04.json`
4. All your Toby collections are now imported!

### Save Your Current Session
1. Press `Alt+S` (or click extension icon â†’ Save Session)
2. Give it a name (optional)
3. Later, restore it from the dashboard

### Search Your Tabs
- **Quick search**: Press Shift twice rapidly
- **Regular search**: Type in the search bar (or press `/`)
- Click any result to open it

## Tips

- **Keyboard shortcuts**:
  - `/` to focus search
  - `Shift + Shift` (double-tap) for quick search
  - `Alt+S` to save session
  - `Esc` to close search

- **Popup actions**: Click the extension icon for quick access

- **Export regularly**: Click "Export" to backup your data

## You're Ready!

The extension is now fully set up. Start organizing your tabs into collections!
